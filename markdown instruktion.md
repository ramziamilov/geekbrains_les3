# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходим ообрамить его звездочками "*"

*курсив* 

Чтобы выделить текст полужирным необходимо обрамить в две звездочки "**"

**полужирный**
